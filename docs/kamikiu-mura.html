Engineering the World

Kamikiu Mura is a 2D adventure game focused on exploration. The core development challenge was creating a responsive, intelligent world that reacts to player movement and actions in real-time.

Technical Implementation

Role: Lead Programmer / Developer

  • Implemented enemy AI using A* pathfinding and finite state machines.
  • Integrated AI behavior with Unity animation and collision systems to ensure frame-perfect hit detection.
  • Engineered modular C# scripts for character controllers, allowing for easily tunable physics and movement parameters.
